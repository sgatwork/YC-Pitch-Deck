# Changelog

## v1.0.0 — April 2026

Initial release.

### Design system
- Off-white (`#F0F0F0`) background on all slides
- Helvetica Bold for headlines, Courier New for all labels and metadata
- White rounded card system with soft shadows as the primary layout element
- Two-column layout: big headline left, evidence cards right
- Klein Blue (`#002FA7`) as default accent — overridable with founder's brand colour
- Logo mark top-right on every slide
- Complete PptxGenJS code patterns for all 7 slide types

### Content framework
- 10-question founder interview including brand colour and logo
- Vertebrae method for identifying the 3–4 most investable facts
- Slide-by-slide format specs matching the design system
- Story architecture guide

### Skill files
- `SKILL.md` — entrypoint with CREATE and REVIEW modes
- `references/content.md` — slide content guide
- `references/design.md` — design system + production code patterns
